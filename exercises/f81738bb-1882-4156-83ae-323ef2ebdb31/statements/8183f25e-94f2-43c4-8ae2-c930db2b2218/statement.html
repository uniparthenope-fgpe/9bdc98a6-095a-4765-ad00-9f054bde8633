Dict to Dataframe

Given a dictionary as input, convert it into a corresponding dataframe and display it.


Example input:

{ 'name': ['Vinay', 'Kushal', 'Aman'],  'age' : [22, 25, 24],  'occ' : ['engineer', 'doctor', 'accountant'] }


Example output:

     name  age         occ

0   Vinay   22    engineer

1  Kushal   25      doctor

2    Aman   24  accountant
